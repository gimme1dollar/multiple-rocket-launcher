# Sphoenix
Multiple Rocket Launching Robot


## Specification
- FDM 3D printed solid rocket engine 
- Launch 6 rockets
- Motor control: P-control algorithm with absolute-encoder
- Vibration damping with rigid couplings
- Communication: RS485 module
- Ignite electrically
